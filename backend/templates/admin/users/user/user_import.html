Bulk User Import Instructions

Upload an Excel file (.xlsx) containing user data with the following columns:

  • email (required): The user's email address, which will be used for login and notifications
  • password (required): The user's initial password (will be hashed automatically)
  • name (required): The user's full name or display name
  • username (optional): A unique username for the user. If not provided, the email address will be used as the username

Note: Ensure all required fields are filled and email addresses are unique. Duplicate emails will result in import errors.
